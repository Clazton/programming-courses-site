Python for Beginners

Introduction

Python is a popular programming language known for its readability and simplicity. Let's learn some basics!

Variables and Data Types

Variables store information. You assign values using the = operator:

x = 5
name = "Alice"
pi = 3.14
is_happy = True

Python data types include:

  • int: integers like 5, -3
  • float: decimal numbers like 3.14
  • str: strings like "hello"
  • bool: True or False

Control Flow: If Statements

Run code conditionally with if:

age = 18
if age >= 18:
    print("You are an adult.")
else:
    print("You are a minor.")

Note the indentation - it defines blocks in Python.

Loops: For and While

Repeat actions using loops.

For loop:

for i in range(5):
    print(i)

While loop:

count = 0
while count < 5:
    print(count)
    count += 1

Functions

Group reusable code with functions:

def greet(name):
    print("Hello, " + name + "!")

greet("Alice")
greet("Bob")
