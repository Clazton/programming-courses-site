Python for Beginners

Welcome to Python! In this course, you'll learn variables, data types, loops, functions, and more.

Variables and Data Types

In Python, variables store data. Example:

x = 5
name = "ChatGPT"
is_active = True

Loops

Loops help us repeat code. Example:

for i in range(5):
    print(i)

Functions

Functions group reusable code:

def greet(name):
    print("Hello " + name)
