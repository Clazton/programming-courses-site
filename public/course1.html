Python Basics

Introduction

Python is a popular high-level programming language known for its readability and versatility.

Let's start with some basic concepts:

  • Variables: Containers for storing data values.
  • Data Types: int, float, string, bool, list, dict, etc.
  • Functions: Blocks of reusable code.

Variables & Data Types

age = 25
name = "Alice"
height = 5.6
is_student = True

In Python, variables don't need explicit declaration of their type.

Functions

def greet(name):
    print(f"Hello, {name}!")

greet("Bob")

This defines a function named greet that prints a greeting.
